#### Instructions ####

1) Compile KOTLIN file, running the following command:
   - kotlinc HelloWorld.kt
   
   - 1.1) It will generate the following file:
          - HelloWorld.exe

2) Run/Execute KOTLIN file, running the following command:
   - ./HelloWorld
   
   - 2.1) It will be printed the following output on the terminal:
          "Hello World!!!"
