#### Instructions ####

1) Compile OCAML file, running the following command:
   - ocaml -o HelloWorld HelloWorld.ml
   
   - 1.1) It will generate the following file:
          - HelloWorld.exe

2) Run/Execute GO file, running the following command:   
   - ./HelloWorld

   - 2.1) It will be printed the following output on the terminal, until you press ENTER or any key:
          "Hello World!!!"
