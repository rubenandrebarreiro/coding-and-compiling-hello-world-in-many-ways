#### Instructions ####

1) Compile CRYSTAL file, running the following command:
   - crystal build HelloWorld.cr
   
   - 1.1) It will generate the following file:
          - HelloWorld

2) Run/Execute CRYSTAL file, running the follwing command:
   - 2.1) ./HelloWorld
   
   - 2.2) It will be printed the following output on the terminal:
          "Hello World!!!"
