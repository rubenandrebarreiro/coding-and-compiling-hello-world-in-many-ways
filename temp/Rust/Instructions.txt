#### Instructions ####

1) Compile RUST file, running the following command:
   - rustc HelloWorld.rs
   
   - 1.1) It will generate the following file:
          - HelloWorld.exe

2) Run/Execute RUST file, just click in it:   
   - ./HelloWorld

   - 2.1) It will be printed the following output on the terminal:
          "Hello World!!!"
