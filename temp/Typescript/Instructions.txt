#### Instructions ####

1) Compile TYPESCRIPT file, running the command:
   - tsc HelloWorld.ts
   
   - 1.1) It will generate the following file:
          - HelloWorld.js

2) Run/Execute TYPESCRIPT file, running the following command:
   - node HelloWorld.js
   
   - 2.1) It will be printed the following output on the terminal:
          "Hello World!!!"

NOTE:
- If you don't have the TYPESCRIPT compiler (tsc) installed, you should run:
  * npm install -g typescript
