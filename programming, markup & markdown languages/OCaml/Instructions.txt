#### Instructions ####

1) Compile OCAML file, running the following command:
   - ocamlc -o HelloWorld HelloWorld.ml
   
   - 1.1) It will generate the following files:
          - HelloWorld, HelloWorld.cmi and HelloWorld.cmo

2) Run/Execute OCAML file, running the following command:   
   - ./HelloWorld

   - 2.1) It will be printed the following output on the terminal:
          "Hello World!!!"
