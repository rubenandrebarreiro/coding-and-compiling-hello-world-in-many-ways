1) Compile VISUAL BASIC file, running the following command:
   - vbc -reference:Microsoft.VisualBasic.dll HelloWorld.vb

          - HelloWorld.exe

   - ./HelloWorld

   - 2.1) It will be printed the following output on the terminal:
          "Hello World!!!"
