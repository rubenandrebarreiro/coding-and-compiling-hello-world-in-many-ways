#### Instructions ####

1) Compile RED file, running the following command:
   - red -c HelloWorld.red
   
   - 1.1) It will generate the following file:
          - libRedRT.so, libRedRT-dfs.r, libRedRT-include.red and HelloWorld

2) Run/Execute RED file, running the follwing command:
   - 2.1) ./HelloWorld
   
   - 2.2) It will be printed the following output on the terminal:
          "Hello World!!!"
