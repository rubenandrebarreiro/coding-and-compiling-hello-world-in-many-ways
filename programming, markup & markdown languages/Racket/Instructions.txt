#### Instructions ####

1) Compile RACKET file, running the following command:
   - raco exe --gui HelloWorld.rkt
   
   - 1.1) It will generate the following file:
          - HelloWorld

2) Run/Execute RACKET file, running the follwing command:
   - 2.1) ./HelloWorld
   
   - 2.2) It will be printed the following output on the terminal:
          "Hello World!!!"
