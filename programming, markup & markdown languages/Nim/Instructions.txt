#### Instructions ####

1) Compile NIM file, running the following command:
   - nim c -r --verbosity:0 HelloWorld.nim
   
   - 1.1) It will generate the following file:
          - HelloWorld

2) Run/Execute NIM file, running the follwing command:
   - 2.1) ./HelloWorld
   
   - 2.2) It will be printed the following output on the terminal:
          "Hello World!!!"
