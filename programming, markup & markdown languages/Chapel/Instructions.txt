#### Instructions ####

1) Compile CHAPEL file, running the following command:
   - chpl -o HelloWorld HelloWorld.chpl
   
   - 1.1) It will generate the following file:
          - HelloWorld

2) Run/Execute CHAPEL file, running the follwing command:
   - 2.1) ./HelloWorld
   
   - 2.2) It will be printed the following output on the terminal:
          "Hello World!!!"
