#### Instructions ####

1) Compile ZIG file, running the following command:
   - zig build-exe HelloWorld.zig 
   
   - 1.1) It will generate the following files:
          - HelloWorld.o and HelloWorld

2) Run/Execute CUDA file, running the follwing command:
   - 2.1) ./HelloWorld
   
   - 2.2) It will be printed the following output on the terminal:
          "Hello World!!!"
