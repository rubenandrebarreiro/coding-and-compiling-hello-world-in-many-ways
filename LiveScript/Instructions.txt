#### Instructions ####

1) Compile LIVESCRIPT file, running the command:
   - lsc -c HelloWorld.ls
   
   - 1.1) It will generate the following file:
          - HelloWorld.js

2) Run/Execute LIVESCRIPT file, running the following command:
   - node HelloWorld.js
   
   - 2.1) It will be printed the following output on the terminal:
          "Hello World!!!"

NOTE:
- If you don't have the LIVESCRIPT compiler (tsc) installed, you should run:
  * npm install -g livescript
